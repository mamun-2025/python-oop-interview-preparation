# 🎯 Topics Covered

🟢 1. Class & Object
- Class = Blueprint / Template
- Object = Instance of class
- Constructor (__init__)
- Instance vs Class variables
- Instance methods

🟡 2. Inheritance
- Code reusability
- Parent → Child relationship
- Types:
- Single
- Multiple
- Multilevel
- Hierarchical
- super() usage
- Method overriding

🔵 3. Encapsulation
- Data hiding
- Access modifiers:
- Public
- Protected (_var)
- Private (__var)
- Getter & Setter
- @property decorator

🟣 4. Polymorphism
- Same method, different behavior
- Method overriding
- Duck typing
- Operator overloading

🔴 5. Abstraction
- Hide implementation details
- Abstract class (ABC)
- Abstract method (@abstractmethod)
- Rules define in parent, implementation in child

🟠 6. Method Overriding vs Overloading
- Overriding → Runtime polymorphism
- Overloading → Python workaround using:
- Default arguments
- *args

🟢 7. Instance vs Class vs Static Method
- Instance method → works with object (self)
- Class method → works with class (cls)
- Static method → utility function

🟡 8. Decorators
- Function wrapper
- Add extra functionality without modifying original code

Uses:
- Logging
- Authentication
- Validation

🔵 9. Generators
- yield keyword
- Lazy evaluation
- Memory efficient

Use cases:
- Large data processing
- Streaming

🟣 10. Iterator vs Iterable
- Iterable → loop করা যায়
- Iterator → next() দিয়ে value দেয়
- iter() vs next()
- Generator = Iterator

# 🔥 Example Code
```bash
class Person:
    def __init__(self, name):
        self.name = name

    def greet(self):
        print("Hello", self.name)

p = Person("Mamun")
p.greet()
```
